# MedDistrib

Dashboard Flask untuk sistem rekomendasi hybrid distribusi tenaga kesehatan.

## Menjalankan project
1. Install dependensi
   pip install -r requirements.txt

2. Jalankan Flask
   python app.py

3. Buka browser
   http://127.0.0.1:5000

## Struktur
- app.py: aplikasi utama Flask
- templates/: file HTML
- static/: CSS dan JS
- model/: modul backend model
- data/: dataset
